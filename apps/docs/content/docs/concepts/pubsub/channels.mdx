# Pub/Sub Channels

Pub/Sub Channels are the core primitive of Erebus. They are used to publish and subscribe to topics on channels.
The design is similar to MQTT with Quality of Service level 1.

## What is avaliable?

- Core (low-level building block)
- React (high-level building block with typesafety and schemas)
