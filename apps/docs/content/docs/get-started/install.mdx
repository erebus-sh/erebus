# Install Erebus

Erebus is available on npm and bun. You can install it using the following commands:

```package-install
@erebus/sdk
```

after installing the package, you can now start following the
- [quickstart guides core](/docs/get-started/core/quickstart-example)
- [quickstart guides next.js](/docs/get-started/next.js/quickstart-example)
